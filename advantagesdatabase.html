advantages database Details
Data management A DBMS provides a mechanism for easier management of the data held by an organisation.
Data independence In a database system the data is held independently of the application programs that use it. Changes to the data or the way it is stored will not necessarily affect the programs that use the data.
Data sharing The collection of data in a central data repository and managed by a DBMS allows the data to be shared by several users and applications.
Reduced maintenance A number of tools for maintaining the data are already built in to most DBMS. Tools may include report generators, query languages, form generators, code checkers, data dictionaries and maintenance logs.
